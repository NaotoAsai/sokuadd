# 収支を漏れなく楽に管理できるアプリ『sokuadd』
## 収支管理をしていると細かな出費などを忘れてしまったりして正確な収支管理ができずに、めんどくさくなってつけるのをやめてしまったり、アプリが使いづらくてつけるのがめんどうになったり、、そういった悩みを解決するために理想の収支管理アプリを形にしてみました。
### 使用技術：Laravel,Nuxt.js
### アプリURL: https://sokuadd.com
### サンプルユーザー
